AQI Frontend Setup Instructions

Requirements:

* Node.js (v16 or above recommended)

Steps to Run:

1. Open terminal in this folder

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev

4. Open browser:
   with link on the vite cmd

Important Notes:

* Ensure backend is running before using the frontend

* Backend should be available at:
  http://127.0.0.1:8000

* If API calls fail, check:

  * Backend server is running
  * Correct API URL is used in frontend

Optional:

* You can configure API URL using environment variables if needed
